## 🚀 Quick Start Guide

### For First-Time Users

**Step 1: Understand the Context** (30 minutes)
1. Read: `workshop-requirements-summary.md`
2. Review the requirements that shaped this design
3. Confirm these still match your situation

**Step 2: Review the Plan** (1 hour)
1. Read: `north-star-workshop-plan.md`
2. Walk through the complete agenda
3. Note any sections to customize for your context

**Step 3: Set Up Pre-Work** (1 week before workshop)
1. Use: `pre-work-template.md`
2. Customize for your team (update timelines, examples)
3. Send to participants with 2-3 day deadline

**Step 4: Build the Mural Board** (2-3 days before workshop)
1. Use: `mural-board-guide.md`
2. Create board frame-by-frame
3. Pre-populate Frame 2 with pre-work synthesis
4. Test all interactive features

**Step 5: Prepare to Facilitate** (1 day before workshop)
1. Read: `facilitator-script.md`
2. Practice key transitions
3. Prepare opening synthesis from pre-work
4. Do tech checks

**Step 6: Facilitate the Workshop** (Workshop day)
1. Use: `facilitator-script.md` as your guide
2. Follow timing closely
3. Capture outputs on Mural
4. Assign next step owners

**Step 7: Document & Follow Up** (Within 48 hours)
1. Use: `post-workshop-deliverables.md`
2. Fill in all outputs from Mural
3. Distribute to participants
4. Schedule follow-up actions

## 🎯 Customization Guide

### If Your Situation Differs

**More time available (full day)?**
- See `north-star-workshop-plan.md` → "Adaptation Notes" section
- Add: deeper gap analysis, customer research review, detailed Year 1 roadmap

**Less time available (2 hours)?**
- See `north-star-workshop-plan.md` → "Adaptation Notes" section
- Focus: Vision + Year 1 themes only; defer capabilities to follow-up

**Larger group (15+ people)?**
- See `north-star-workshop-plan.md` → "Adaptation Notes" section
- Add: co-facilitator, more breakout groups, extended report-out times

**Different format (fully remote or in-person)?**
- Remote: Add virtual energizers, more frequent breaks
- In-person: Use physical materials, wall space for boards

**No existing strategy materials?**
- Adjust pre-work to remove "review existing materials" section
- Add 15 min to Phase 2 for foundational vision work

## 💡 Pro Tips

### For First-Time Facilitators
1. **Practice the script** out loud at least once
2. **Test all tech** the day before (Mural, Teams, breakouts)
3. **Time yourself** on key transitions
4. **Have a co-facilitator** for tech support if possible
5. **Trust the process** - the structure will hold you

### For Experienced Facilitators
1. **Adapt to the room** - if energy is high, let discussion run; if low, add energizer
2. **Watch for non-participation** - actively invite quiet voices
3. **Use parking lot liberally** - protect the agenda
4. **Read the underlying dynamics** - surface tensions productively
5. **Balance structure with flexibility** - know when to deviate

### For Hybrid Facilitation
1. **Center the Mural** - it's the equalizer between remote and in-person
2. **Assign a tech shepherd** in the room to help remote folks
3. **Over-communicate** - describe what you're doing visually
4. **Check in frequently** with remote participants
5. **Use breakouts** to create intimacy that large hybrid rooms lack

## 🆘 Common Questions

**Q: What if pre-work isn't completed?**
A: Build in 15 min at start for rapid pre-work completion. Cut gallery walk to compensate. See Facilitator Script → "Emergency Scenarios"

**Q: What if we can't reach consensus on vision?**
A: Use "disagree and commit" - identify decision-maker, make the call, move forward. Document dissent in parking lot.

**Q: What if we run over time?**
A: See Facilitator Script → "Emergency Scenarios" → "Running severely over time"

**Q: What if tech fails?**
A: Switch to backup (Google Jamboard or Doc). Simplify to text capture. See Facilitator Script → "Tech Troubleshooting"

**Q: What if outputs feel too rough/incomplete?**
A: That's expected! Workshop creates drafts. Assigned owners refine. Set expectation at start.

**Q: Can I use this for a different type of workshop?**
A: Yes! The structure (Connect → Envision → Plan → Act) works for many strategic workshops. Adapt the content.
